# pinger
Simple node app to continuously ping a list of addresses.

## How to use:

- git clone
- npm install
- Add URL's to the sites array.
```
const sites = [
    'http://google.com',
    'http://yahoo.com'
];
```
- Change the minutes variable to control the frequency in minutes.
```
const MINUTES = 25;
```
- npm start
